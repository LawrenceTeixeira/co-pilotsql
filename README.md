# Setup

## Set your environment variables
```
OPENAI_API_KEY=
OPENAI_MODEL=
VANNA_MODEL=
VANNA_API_KEY=
AZURESQL_SERVER=
AZURESQL_DATABASE=
AZURESQL_USERNAME=
AZURESQL_PASSWORD=

```

## Install dependencies
```
pip install -r requirements.txt
```

## Run the server
```
python app.py
```
